## 作成履歴 API

### 作成履歴作成

| 項目             | 内容                                                        |
| ---------------- | ----------------------------------------------------------- |
| メソッド         | POST                                                        |
| エンドポイント   | /creation_history                                           |
| リクエストボディ | `{ "userId": ユーザーの ID, "cocktailId": カクテルの ID }`  |
| レスポンス       | 201：正常終了<br>400：リクエスト不正<br>500：サーバーエラー |

### 作成履歴取得

| 項目                 | 内容                                                        |
| -------------------- | ----------------------------------------------------------- |
| メソッド             | GET                                                         |
| エンドポイント       | /creation_history                                           |
| リクエストパラメータ | `{ "cocktailId": Int, "userId": Int }`                      |
| レスポンス           | 200：正常終了<br>400：リクエスト不正<br>500：サーバーエラー |
